# Python Utility Projects Collection

This repository contains a set of beginner-to-intermediate level Python projects designed to demonstrate practical programming skills, data handling, and user interaction through command-line interfaces. Each project focuses on a specific use case and showcases modular design, input validation, and basic data persistence.

## üì¶ Projects Included

### 1. Contact Book
A simple contact management system that allows users to:
- Add, search, update, and delete contact entries
- Store contact details such as name, phone number, and email
- Save data locally using file handling or database

### 2. Employee Management System
A basic system to manage employee records:
- Add new employees with details like ID, name, department, and salary
- Update or delete employee information
- View all employee records
- Useful for learning CRUD operations and structured data handling

### 3. Marks Calculator
A utility to calculate student marks and performance:
- Input subject-wise marks
- Calculate total, average, and grade
- Display results in a readable format
- Great for practicing conditional logic and arithmetic operations

### 4. Notes Manager
A lightweight notes application:
- Create, view, edit, and delete notes
- Organize notes by title or date
- Save notes persistently using text files or simple databases
